Simple Calculator

This is a simple calculator built with HTML, CSS, and JavaScript. Where you can perform basic arithmetic operations like addition, subtraction, multiplication, and division. It features a clean and modern design with a responsive layout, making it easy to use on both desktop and mobile devices. The calculator includes a display for showing the current input and results, as well as buttons for digits, operations, and clearing the display. Whether you need to quickly calculate something or just want to have fun with a simple tool, this calculator is here to help!
